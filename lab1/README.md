# Лабораторная работа
## Предобработка данных и визуализация (датасет insurance.csv)

### **Описание данных**

В данной работе используется датасет `insurance.csv`, содержащий информацию о страховых расходах клиентов медицинской страховой системы.  
Каждая строка представляет одного клиента.  

Состав признаков:

- **age** — возраст клиента (числовой тип)  
- **sex** — пол (`male` / `female`)  
- **bmi** — индекс массы тела (числовой тип)  
- **children** — количество детей на иждивении (целочисленный тип)  
- **smoker** — факт курения (`yes` / `no`)  
- **region** — регион проживания (`southwest`, `southeast`, `northwest`, `northeast`)  
- **charges** — затраты на медицинскую страховку (числовой тип, целевая переменная)

### **Подключённые библиотеки**

В ноутбуке используются следующие библиотеки:

- `pandas` — загрузка данных, обработка таблиц  
- `numpy` — базовые вычисления  
- `matplotlib.pyplot` — визуализация данных  
- `seaborn` — продвинутые графики и тепловые карты  
- `sklearn` (по желанию) — обработка категориальных признаков, нормализация

### **Предобработка данных**

В ноутбуке были выполнены следующие процедуры:

- загрузка данных из CSV;  
- проверка структуры таблицы с помощью `info()` и `head()`;  
- анализ типов данных и автоматическое приведение категориальных признаков к типу `category`;  
- поиск пропусков и проверка целостности данных (`isnull()`, `describe()`);  
- выявление возможных выбросов с помощью boxplot-графиков;  
- преобразование категориальных признаков для последующей обработки (label encoding или one-hot encoding — по необходимости);  
- статистический анализ распределений числовых столбцов.

### **Визуализации, построенные в ходе анализа**

- гистограммы распределения возраста, расходов, BMI;  
- boxplot графики по категориям (например, `charges` по `smoker`);  
- тепловая карта корреляции числовых признаков;  
- сравнение средней стоимости страховки в зависимости от пола, курения и региона.

### **Содержимое папки lab1**

- `insurance.csv` — основной файл данных;  
- `preprocessing.ipynb` — ноутбук с кодом предобработки и визуализаций;  
- `README.md` — описание используемых данных и методов анализа.
